## Getting Started

### First Time Login

1. Navigate to the platform homepage
2. Click the **Login** button in the navigation bar
3. Select your role (Surveyor or Admin)
4. Enter your credentials
5. Click **Login**

## For Surveyors

### Starting a New Survey

1. Login with your surveyor credentials
2. From your dashboard, click **Start New Survey**
3. Select **Form A - Individual Survey**
4. Complete all required fields
5. Progress through Forms B and C sequentially

### Dashboard Features

Your dashboard shows:
- **Today's Progress**: Circular progress indicator
- **Weekly Stats**: Total surveys this week
- **Monthly Total**: Cumulative monthly count
- **Recent Surveys**: Table of your latest submissions
- **Assigned Areas**: Your designated survey regions

### Survey Workflow

```
Form A (Individual) → Form B (Group) → Form C (Leader) → Corpus Questions → Word List
```

Each form must be completed before accessing the next.

### Daily Targets

- Your daily target is displayed on the dashboard
- Complete surveys count toward your target
- Progress resets daily at midnight

## Using Speech Recognition

### Setup

1. Allow microphone access when prompted
2. Select your preferred language
3. Click the microphone icon
4. Speak clearly

### Language Options

| Language | Code | Script |
|----------|------|--------|
| Urdu | ur-PK | اردو |
| English | en-US | English |
| Punjabi | pa-PK | ਪੰਜਾਬੀ |
| Sindhi | sd-PK | سنڌي |
| Pashto | ps-PK | پښتو |

### Voice Input Process

1. **Click Mic Button**: Turns red when recording
2. **Speak Clearly**: Wait for transcription
3. **Click Again**: Stop recording
4. **Review Text**: Edit if necessary
5. **Continue**: Move to next field

### Tips for Better Recognition

- Speak in a quiet environment
- Use headset microphone if available
- Speak at normal pace
- Pronounce words clearly
- Pause between sentences

## Form B - Group Discussions

### Setting Participant Count

1. Enter number of participants (1-20)
2. System creates individual sections
3. Each person gets separate response fields

### Recording Individual Responses

For each participant:
1. Enter their name
2. Record response to each question
3. Use voice input in their language
4. Save before moving to next person

### Question Structure

Each participant answers:
- Q1: Mother tongue occasions
- Q2: Wider communication language
- Q3: Language regions
- Q4: Similar languages
- Q5: Top community languages
- Q6: Cultural pride moments
- Q7: Language dreams

### Group Consensus

After individual responses:
1. Optional summary section
2. Record overall observations
3. Note any disagreements
4. Submit complete form

## Troubleshooting

### Speech Recognition Not Working

**Problem**: Microphone button not responding
**Solutions**:
- Check browser microphone permissions
- Ensure HTTPS connection
- Try Chrome browser (best support)
- Refresh page and try again

### Login Issues

**Problem**: Cannot login with credentials
**Solutions**:
- Verify email format
- Check password (case-sensitive)
- Use demo credentials for testing
- Clear browser cache

### Form Submission Errors

**Problem**: Form won't submit
**Solutions**:
- Check all required fields (marked with *)
- Ensure internet connection
- Save draft and retry
- Contact support if persists

### Browser Compatibility

**Recommended Browsers**:
- Chrome 90+ ✅
- Firefox 85+ ✅
- Safari 14+ ⚠️ (limited speech support)
- Edge 90+ ✅

### Data Not Saving

**Problem**: Responses disappear
**Solutions**:
- Check browser local storage
- Don't use incognito mode
- Save frequently
- Export data periodically

## Best Practices

### For Quality Data

1. **Verify Information**: Double-check entries
2. **Complete All Fields**: Don't skip optional fields
3. **Use Native Language**: Respondent's mother tongue
4. **Record Accurately**: Preserve original pronunciation
5. **Add Context**: Use comment fields

### For Efficiency

1. **Prepare Questions**: Review before meeting respondents
2. **Test Equipment**: Check microphone before field work
3. **Batch Submissions**: Submit multiple surveys together
4. **Regular Syncing**: Upload data daily
5. **Backup Locally**: Keep offline copies

### Privacy & Ethics

1. **Get Consent**: Always obtain permission
2. **Protect Identity**: Use codes if requested
3. **Respect Culture**: Be sensitive to local customs
4. **Secure Data**: Don't share login credentials
5. **Report Issues**: Notify admin of problems

## Frequently Asked Questions

**Q: Can I save partially completed surveys?**
A: Yes, data is auto-saved in your browser. Complete later from same device.

**Q: How many surveys can I submit daily?**
A: No limit, but you have a target of 10 surveys per day.

**Q: Can I edit submitted surveys?**
A: No, submissions are final. Double-check before submitting.

**Q: Is offline mode available?**
A: Currently no, but it's planned for future updates.

**Q: Can I use mobile devices?**
A: Yes, the platform is mobile-responsive, but desktop is recommended for data entry.
